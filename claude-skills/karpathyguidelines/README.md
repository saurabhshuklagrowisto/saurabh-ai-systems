# karpathyguidelines

A Claude Code skill that encodes four behavioral guidelines for reducing common LLM coding mistakes: think before coding (surface assumptions, ask when unclear), simplicity first (no speculative abstractions), surgical changes (touch only what the task requires), and goal-driven execution (define verifiable success criteria).

Derived from [Andrej Karpathy's observations](https://x.com/karpathy/status/2015883857489522876) on LLM coding pitfalls.

**Source:** [forrestchang/andrej-karpathy-skills](https://github.com/forrestchang/andrej-karpathy-skills) (MIT License). Reproduced here unmodified for convenience.

## Install

Copy `SKILL.md` into `~/.claude/skills/karpathyguidelines/` for a personal (account-level) Claude Code skill, or into a repo's `.claude/skills/` for a project-scoped one.
