# Proof Clay Table Screenshot

Add the Clay table screenshot here as `clay-screenshot.png`.

**What to capture:**
- Full Clay table with all columns visible
- Rows: at least 8–10 branded rows visible
- Columns that must show: Platform, Monthly Traffic, Employee Count, Replatform Signal, Industry, ICP Score, Tier, Hook (AI-generated line)

**Before exporting:**
- Blur or remove company names and domain column (use screenshot editor)
- All enrichment data, scores, tiers and hooks can remain visible

**How to export from Clay:**
1. Open the Clay table
2. Screenshot the full table (Windows: Win + Shift + S, then crop)
3. Use any screenshot editor to blur the Company Name and Domain columns
4. Save as `clay-screenshot.png` and drop it in this folder
