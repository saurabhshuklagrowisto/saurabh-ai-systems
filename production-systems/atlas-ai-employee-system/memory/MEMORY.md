# Company Memory — Index

Central knowledge base for all AI employees. Every employee reads context from here and writes learnings back. The Librarian owns consolidation.

## Directories

- [icp-and-scoring/](icp-and-scoring/) — who we sell to, the 0–100 job-post scoring rubric, target job titles, decision-maker matrix. **Source of truth for Scout.**
- [email-templates/](email-templates/) — master outreach sequences + winning copy variants with performance notes. **Source of truth for Scribe.**
- [brand-voice/](brand-voice/) — tone, vocabulary, positioning, do/don't. **Source of truth for Publicist.**
- [case-studies/](case-studies/) — client proof: placements, savings numbers, testimonials.
- [competitor-intel/](competitor-intel/) — competitor staffing agencies, their content hooks, ad angles (Publicist writes here weekly).
- [lead-history/](lead-history/) — per-company touch history and outcomes (Scout/Scribe append; suppression source).
- [playbooks/](playbooks/) — SOPs: deliverability rules, approval gates, escalation.
- [experiment-log/](experiment-log/) — every test (subject lines, hooks, formats) with result. Check before repeating an experiment.

## Rules

1. One fact, one place — link instead of duplicating.
2. Date every entry (YYYY-MM-DD).
3. Outcomes over opinions — log numbers (open %, reply %, booked calls).
4. Librarian consolidates weekly: dedupe, flag stale facts, update this index.
