# Decision-Maker Matrix (Clay people-lookup targets)

Updated: 2026-07-10 · Keyed to practice size guessed by Scout. Keep lookups tight — max 2 contacts for small, 3 for mid — to stretch Clay credits.

## Solo / small practice (1–5 providers)
The owner IS the decision-maker. One or two contacts max.
1. Owner / Practice Owner / Physician Owner / MD Owner / Dentist Owner
2. Office Manager / Medical Office Manager / Practice Manager

## Mid-size practice / group (6–25 providers, multi-location)
Hiring is delegated to an admin/ops layer.
1. Practice Administrator
2. Practice Manager / Practice Operations Manager
3. Office Manager / Front Office Manager
4. Director of Operations / Operations Manager
5. Billing Manager / Revenue Cycle Manager — prioritize when the posting is billing/RCM-heavy

## Billing / RCM company
1. Owner / CEO / President
2. Director of Operations
3. Client Services Manager

## Per-lead fields to capture
Name · Job Title · LinkedIn URL · Verified Email · Phone · Company (name, website, location, # facilities, # staff, est. revenue)
